# Contributing Learning Resources

Contributions should help learners understand security concepts safely and responsibly.

## Accepted Content

- Beginner-friendly explanations.
- Controlled lab setup notes.
- Defensive validation checklists.
- Secure development guides.
- Accessibility improvements.

## Not Accepted

- Unauthorized testing instructions.
- Live-target walkthroughs.
- Credential theft examples.
- Payload obfuscation guides.
- Private or sensitive implementation details.
