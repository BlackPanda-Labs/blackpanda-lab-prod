# BlackPanda Community

Public community space for BlackPanda Labs roadmap notes, public events, contributor onboarding and volunteer tasks.

BlackPanda Labs is an independent, student-led open-source cybersecurity education and research community focused on authorized security testing, controlled lab learning, secure development and collaborative technical education.

## Community Focus

- Open-source collaboration.
- Beginner-friendly technical learning.
- Documentation and accessibility improvements.
- Secure development education.
- Authorized lab learning content.
- Community build sprints and learning sessions.

This repository should not contain private production source code, secrets, admin links or internal infrastructure details.
