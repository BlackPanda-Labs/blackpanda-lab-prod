# BlackPanda Labs Open-Source Build Sprint #1

## Purpose

A beginner-friendly online collaboration sprint focused on improving open-source projects, documentation, accessibility and safe cybersecurity-learning resources.

## Tracks

- Documentation
- Frontend Improvements
- Accessibility
- Community Onboarding
- Safe Learning Resources
- Open-Source Tooling

## Rules

- Work only on authorized repositories and content.
- Do not test third-party systems.
- Do not submit secrets, stolen data or credentials.
- Submit work through pull requests.
- Follow the code of conduct.
- Explain what changed and how it was tested.

## Recognition

Participants may receive community acknowledgements for completed and reviewed contributions.
