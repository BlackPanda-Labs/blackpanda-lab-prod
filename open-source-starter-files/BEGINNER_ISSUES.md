# Beginner-Friendly Issue Ideas

## 1. Improve Contributor Onboarding README

Difficulty: Beginner
Estimated scope: 1-2 hours
Repository: blackpanda-community

Description: Rewrite the onboarding section so new contributors understand where to start, what repositories exist and how to ask for help.

Acceptance criteria:

- Adds a clear first-time contributor path.
- Links only to public community resources.
- Explains how to find good first issues.

Safe scope note: Do not include private repository links, admin routes or internal maintainer processes.
